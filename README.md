# 🎓 GPA & CGPA Calculator

A simple and responsive GPA & CGPA Calculator built using **HTML, CSS, and JavaScript**. It allows students to calculate semester GPA, track CGPA across multiple semesters, and estimate future CGPA using a what-if analysis.

## Features

- Add, edit, and delete subjects
- Calculate Semester GPA
- Calculate Overall CGPA
- Multi-semester support
- Store previous semester GPAs
- What-if CGPA prediction
- Local storage support (data persists after refresh)
- Displays total subjects and total credits
- Responsive and user-friendly interface

## Tech Stack

- HTML5
- CSS3
- JavaScript (Vanilla JS)
- LocalStorage
